📊 Retail Sales Dashboard with Predictive Insights (Power BI)

This project focuses on analyzing retail sales data and building an interactive Power BI dashboard to uncover key business insights and trends.

📌 Project Overview

The goal of this project is to analyze sales performance across different products, categories, and regions, and present insights through a dynamic dashboard.

📁 Dataset Information

The dataset includes:
Order ID,
Date,
Product Category,
Product Name,
Units Sold,
Unit Price,
Total Sale,
Customer Region

🛠️ Tools Used

Power BI
Excel/CSV Dataset

📊 Key Features of Dashboard

Sales trend analysis over time
Revenue breakdown by product category
Region-wise sales performance
Top-selling products identification
Interactive filters for dynamic exploration
Predictive insights using time-based trends

📈 Insights Gained

Identified high-performing product categories
Analyzed regional sales distribution
Discovered sales trends and seasonal patterns
Enabled better business decision-making through data visualization

💡 Learning Outcomes

Built interactive dashboards using Power BI
Improved data visualization and storytelling skills
Learned how to extract insights from sales data
Gained hands-on experience with business intelligence tools
